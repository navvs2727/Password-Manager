# Secure Password Manager (Python)

A simple command line password manager built with Python using Random Password Generation and File Handling.


## Features

- Generate strong passwords
- Save passwords
- View all saved passwords
- Search password
- Update password
- Delete password

## Technologies Used

- Python
- Random module
- File Handling

## Project Structure

main.py                   -> Main program that runs the application
password_generator.py     -> Generates strong random passwords
password_manager.py       -> Handles CRUD operations for passwords
data.txt                  -> Stores saved passwords
README.md                 -> Project documentation


## Installation and Setup

Clone the repository

git clone https://github.com/navvs2727/Password-Manager.git

## Run the program

python main.py


## Example Menu

After running the program you will see the menu:

1 Generate Password
2 Save Password
3 View Passwords
4 Search Password
5 Update Password
6 Delete Password
7 Exit

Choose the option you want and follow the instructions.
